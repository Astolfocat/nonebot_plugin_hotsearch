## 📖 介绍

基于Nonebot 2.0, onebot v11的热搜获取插件~

可以获取微博、百度、知乎、贴吧、b站五个平台的最新热搜，并以合并转发的形式发送出来~

## 💿 安装


- 使用 nb-cli 安装

```
nb plugin install nonebot-plugin-hotsearch
```

- 使用 pip 安装

```
pip install nonebot-plugin-hotsearch
```
